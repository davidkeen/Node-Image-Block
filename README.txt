Node Image Block 2
==================

Node Image Block 2 is a Drupal module that provides a block that displays all
images attached to a node.


Requirements
------------

To install Node Image Block 2 you need:

    * Drupal 6.x
    * Upload module


Installation
------------

Install Node Image Block 2 as follows:

1. Enable and configure the Upload module, following the instructions for that
   module <http://drupal.org/handbook/modules/upload>.

2. Download the latest stable version of Node Image Block 2 from its project
   page <http://drupal.org/node/48676>.

3. Unpack the downloaded file into sites/all/modules or the modules directory
   of your site.

4. Go to Administer » Site building » Modules, and enable the module.


Configuration, usage and theming
--------------------------------

For information about configuration, usage and theming of Node Image Block 2,
please see the online help provided by the module. Go to Administer » Help, and
click on the "Node Image Block 2" link.
